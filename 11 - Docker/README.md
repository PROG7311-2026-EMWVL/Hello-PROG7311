# 📝 Learning Docker

## 📌 Scenario:  
Here is the flow:
1. We will begin to dockerise using the MathApp, which will connect to your local SQL database.
1. We will then learn DockerCompose to:
    1. Dockerize the MathAPI  in its own container
    1. Dockerize the MathAPIClient  in its own container
    1. Dockerize the SQL database in its own container
    1. Place all contain in a single virtual network to allow them to communicate.
